# elemate-weather
A short technical test for Elemate technologies company using Vue 3 & Typescript, installed via CLI.

## Project setup
```
npm install
```
Enter API key sent by email in .env file
```
VUE_APP_API_KEY=
```

### Run server
```
npm run serve
```
